#### Install dpendencies
```
npm install
```
#### Run the software locally
```
npm run dev
```
#### build for linux
```
npm run build:linux
```
#### build for windows
```
npm run build:win
```
#### build for mac os
```
npm run build:mac
```

- After building the app, navigate to the dist folder and you'll find an executable
